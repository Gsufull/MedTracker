# MedTracker — Розумний менеджер домашньої аптечки

**MedTracker** — це сучасний десктопний застосунок на базі .NET 8 та WPF, розроблений для зручного обліку лікарських засобів, контролю їх термінів придатності та планування графіку прийому. Проєкт реалізований з використанням архітектурного патерну **MVVM**, що забезпечує чітке розділення логіки та інтерфейсу.

## 🌟 Основні можливості

* **Управління інвентарем**: додавання, редагування та видалення медикаментів (назва, категорія, кількість, термін придатності).
* **Інтелектуальні сповіщення**: автоматичне підсвічування препаратів, термін придатності яких минув.
* **Система нагадувань**: створення персонального графіку прийому ліків із зазначенням часу та дозування.
* **Аналітична статистика**: візуалізація даних про загальну кількість ліків, категорії та критичні залишки.
* **Безпека даних**: автентифікація користувачів із шифруванням паролів за стандартом SHA-256.
* **Довідник першої допомоги**: вбудована база інструкцій для надання допомоги у критичних ситуаціях.
* **Персоналізація**: підтримка темної та світлої тем, а також багатомовність (UA/EN).

## 🛠 Технологічний стек

* **Платформа**: .NET 8.0 (Windows Desktop)
* **Інтерфейс**: WPF (Windows Presentation Foundation)
* **Мова програмування**: C# 12
* **Архітектура**: MVVM (Model-View-ViewModel)
* **Зберігання даних**: JSON (локальна база даних)
* **Безпека**: System.Security.Cryptography (SHA-256)

## 📁 Структура проєкту

Проєкт має модульну структуру для полегшення підтримки та масштабування:

* `Models/` — класи даних (`Medicine`, `User`, `Reminder`).
* `ViewModels/` — логіка взаємодії між даними та інтерфейсом.
* `Views/` — XAML-сторінки та вікна (MainWindow, InventoryPage, StatisticsPage тощо).
* `Services/` — сервіси для роботи з файлами, сесіями та логуванням.
* `Helpers/` — допоміжні класи (наприклад, для хешування паролів).
* `Resources/` — словники локалізації (UA/EN) та стилі тем.

## 🚀 Початок роботи

### Вимоги
* Встановлена ОС Windows 10/11.
* Встановлений [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).

### Встановлення та запуск
1.  Клонуйте репозиторій:
    ```bash
    git clone [https://github.com/your-username/medtracker.git](https://github.com/your-username/medtracker.git)
    ```
2.  Відкрийте файл `MedTracker.sln` у Visual Studio 2022.
3.  Відновіть NuGet-пакети.
4.  Натисніть **F5** для запуску проєкту.
